說好不再流淚

清醒的時間大概都是處在悲傷的情緒之中哭泣，或者麻木做的無意義的事。

每天的清晨 5:00 跟 下午 5:00 都習慣性地看向手機，因為我知道我媽會跟我 update 她目前用藥的狀況，以及叫我不要擔心。總是說她在醫院很好，讓我在這邊好好工作照顧好一家人。不過從今天開始當我看向手機的時候，沒有任何新進來的訊息，我明白老媽再也不會給我消息了。我剩下的就是跟她的回憶，也沒辦法再抱抱她，說她好矮！

我妹給我發了一個訊息，說，老媽之前跟她去散步說過的話。

以前總想到美國念書，想要看一看這個令人著迷的世界。因為我覺得我可以而且想要更進一步，爬得更高看到更棒的風景，同時也希望自己更有能力之後，給家人更好的生活。但我知道那時候因為家境不好，所以不可能。也不想讓我媽有太大的負擔，所以這個夢想一直放在心裡。

我忘不了我大學即將畢業的時候我媽問過我，我拿書卷畢業想不想去美國念研究所？她就算借錢也讓我去。我當然想，但我跟她說：我以後可以自己來，我就不信交大有輸其他美國研究所。唸書的時候專挑最硬的課，接很多案子，做最硬的專案。因為我知道，這樣可以最大的提升自己，讓自己變得更強，這樣我才有辦法讓家人過得更好。

結果就在好像開始走 Happy Path 了，妳就永遠的離開了。妳不要老是擔心我們，死肥宅的承受能力很強，妳要多關心妳自己，別像個老媽子一樣碎碎念了！好好愛自己，買好的東西給自己享受，不要每次一回家就叫我帶一堆吃的，水果什麼的回台北了！我可以想像妳的樣子，一定又會叫我不要幹幹叫了。

在醫院度過了妳的60歲生日，妳說的你願望是希望明年可以在家過生日。想不到這個願望還沒實現，而在母親節前夕妳就這樣離開了，老妹說妳已經很努力很努力了。我知道妳真的很努力了，真的辛苦了，從此不再有病痛可以自由自在地去妳想去的地方了！只是不能再跟妳說生日快樂，母親節快樂，新年快樂！也沒辦法再帶妳去吃好東西了，再也沒辦法帶妳去好玩的地方玩了，再也沒辦法摟著妳說老媽妳很弱耶！剩下的，只有回憶了。

老媽請不要忘了我，我會很想妳！生日快樂！母親節快樂！
